# zensen-pdf-viewer

A PDF viewer component for LitElement.

## Features

- Displays PDFs
- Provides inputs to quickly jump between pages
- Supports zooming in
- Displays all pages at once

## Install

Using `npm`:

```
$ npm install @zensen/pdf-viewer
```

Using `yarn`:

```
$ yarn add @zensen/pdf-viewer
```
